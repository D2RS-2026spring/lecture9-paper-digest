什么是 Claude Code？

Claude Code 是 Anthropic 推出的 AI 编程助手，能够：

  • 理解大型代码库，进行智能代码导航
  • 自动编辑文件、执行命令、编写代码
  • 支持 Agent 模式，自主完成任务
  • 与 VS Code 等 IDE 深度集成
